# Loading-Spinners

### ✨ About

Three loading spinners built with pure CSS.
A fun project i made while learning CSS animations. It's really cool to see just how many animations you can create with just a few lines of CSS, no JavaScript needed at all.

---

### 💡 What i learned

- `border-top-color` — sets a color for just one side of a border, wich creates the spinning illusion
- `<span>` — lightweight inline element, great for small pieces like dots and bars
- `infinite` — makes the animation loop forever instead of playing once
- `nth-child(n)` — targets a child element by position, used to set a different
- `animation-delay` — on each dot/bar
- `transform: scaleY()` — stretches or squishes an element vertically (`0` =  flat, `1` = normal, `1.5` = taller)
